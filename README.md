# Word Guess Game
This project is a browser-based word guessing game inspired by Wordle. The player attempts to guess a hidden 5-letter word within six attempts using keyboard input. After each guess, the game provides feedback to help guide the player.
## Features
 - 6×5 game board layout
 - Keyboard input (letters, Backspace, Enter)
 - Randomly selected hidden word
 - Letter feedback system (correct, present, absent)
 - Game states: playing, win, lose
 - Restart functionality
 - Current input indicator
 - Tile bounce animation on input
## How to Play
 - Type a 5-letter word using your keyboard
 - Press Enter to submit your guess
 - Use the feedback to improve your next guess
 - You have 6 attempts to guess the correct word
 - Click “Restart Game” to play again
